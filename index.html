Installation Guide

Follow our quick setup steps to install and start using MusicSpot instantly.

Install Git

Download Git for your operating system from the official website, for a detailed installation walkthrough, you can also watch this YouTube tutorial

Verify Git installation

Open Command Prompt or PowerShell and run the following command:

git --version
Python installation

Download Python for your operating system from official website, for a detailed installation walkthrough, you can also watch this YouTube tutorial

Verify Python installation

Open Command Prompt or PowerShell and run the following command:

python -V
